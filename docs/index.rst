Welcome to Spectral View's documentation!
=========================================

About
-----

Spectral View is web application for interactive star spectra viewing. It is
built using `Tornado <http://www.tornadoweb.org/>`_ web framework on top of
`matplotlib <http://matplotlib.org/>`_ and
`Motor <https://motor.readthedocs.io/>`_ which is driver for MongoDB.
